# ALXprodev-advanced_git

This repository demonstrates the use of GitFlow for managing feature development, releases, and hotfixes.

## GitFlow Branches

- **main**: Stable production-ready code
- **develop**: Main development branch
- **feature/**: Feature branches for new features
- **release/**: Branches for preparing production releases
- **hotfix/**: Branches for quick production fixes

## Setup Instructions

To initialize GitFlow with default settings, run:

```bash
git flow init -d
